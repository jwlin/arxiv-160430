Welcome to New York University’s Online Admissions Application System!

Please log-in or create a new user account below.

Notes:

  • Please do not use any existing NYU University ID (N#), NYU NetID, or NYU email for this application
    • This Application User ID will only be used to access your online admissions application
    • You will be able to provide an NYU email address for correspondence later in the application process.
  • Passwords must be 8 characters long and contain at least one special character (!,@,#,$, etc) and one number (0-9).
    • If you get a message stating that the user could not be authenticated, this means that you have provided an incorrect User ID and/or password. Recall that passwords are case sensitive.
  • If you are unable to login, please use the ‘Forgot your User ID?’ and ‘Forgot your password?’ buttons.
    • If you attempt to retrieve your User ID or reset the password and the system is not able to locate your email address, this means that the system has temporarily locked out your account for security reasons. In this case, do not attempt to login but reset the password using the ‘Forgot your password?’ functionality on the login page after 5 minutes.

If paying the application fee poses a financial hardship for you, we don’t want that to stop you from applying. To request a fee waiver, e-mail engineering.gradinfo@nyu.edu stating that you would like to request your fee be waived. No other documentation is required.

For technical issues, contact the IT Service Desk in a number of ways:

- For help by telephone, call 1-212-998-3333 to speak with IT Service Desk staff 24 hours a day, seven days a week.
- For help online,visit http://www.nyu.edu/its/askits/helpdesk/
- For in-person help, visit the IT Service Desk at 10 Astor Place, 4th floor, Monday through Friday, 9:00am to 6:00pm ET.
