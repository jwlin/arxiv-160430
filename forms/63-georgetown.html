Welcome to Georgetown University Graduate School's Online Application

Before you apply, consult the Graduate School webpage for information on academic programs and application deadlines. For additional information, please go to the application instructions page.

  • Apply for Admission
    • Create an account. You will then have login credentials that will enable you to complete your application over several sessions. You will access your secure application in subsequent sessions by signing in under “User Log in.” If you have submitted an application previously, you must create a new account for the new application. The system will prompt you on how to complete the application. Once satisfied with the application, submit it online to Georgetown.
  • Apply for Admission to Multiple or Dual Programs
    • Create an account for each program of interest, complete each application, and submit accordingly.
  • Online Submission of Application Materials
    • Online submission of all application materials is required, except for official test scores and official academic transcripts. Official test scores should be requested from testing organizations. Official transcripts should be requested from all academic institutions attended. These should be mailed directly from the institution, or can be sent via secure online delivery if your institution provides that option. Please DO NOT send unofficial copies. Please also DO NOT send hard copy versions of documents submitted online. Your application information is maintained on a secure server and remains confidential prior to submission. Use the Technical Support link for help or assistance during the completion and submission process.
  • Online Review of the Application
    • Your application will be reviewed online by the appropriate admissions committee once submitted to Georgetown University. All required application materials must be received by the appropriate application deadline. Once a decision is made by your program's committee, the Graduate School will confirm this decision and will send an official online letter on behalf of the Dean.
  • Check the Status of Your Application
    • Check the status of your application by logging into this system with the PIN and password assigned to you for the application. Additionally, you will receive important email messages regarding your application status.
  • Questions?
    • Use the links provided above to get more information on academic program offerings, as well as application instructions and deadlines.

System Requirements

Application System Requirements

Supported browsers include Internet Explorer 9 and 11, Firefox, Chrome and Safari.

While the majority of features will work with other browsers, we can only guarantee full compatibility and offer support for the browsers mentioned above.

If you are using one of the supported browsers and are experiencing problems, we encourage you to upgrade to the latest version. If you still experience problems after upgrading, please make sure that your browser is set to accept cookies. Also, make sure the following browser settings are enabled:

  • JavaScript must be enabled.
  • Popup blockers must be disabled.

If your browser has a section for trusted sites, please enter these two URLs:

  • http://app.applyyourself.com
  • https://app.applyyourself.com

You will need Adobe Reader 6.0 or higher to download, view and print PDF files. While PDF files should work with most other PDF viewers, we can only guarantee full compatibility and support for Adobe Reader 6.0 or higher.
