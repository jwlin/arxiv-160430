Welcome

Welcome to UCR's Graduate Applicant Login Page. Once you have logged in, you will be able to complete and submit a UCR Graduate Application, review the status of your application, and view and respond to UCR admission decisions.

If this is the first time you have visited this web page, please create a UCR Graduate Applicant Identity (a username and password). Your username must be your e-mail address. We suggest you create a password that is both hard to guess and easy to remember. Please note that UCR's web-enabled Graduate Application makes extensive use of e-mail, and it is therefore very important you provide an e-mail address that will allow us to communicate with you electronically.

If you have already created a UCR Graduate Applicant Identity, please login to access UCR's application web site.

Please note that items on this page that are presented in red italics are required fields.

Please note that due to budgetary restrictions, some programs are unable to admit new students to mid-year terms (Winter and Spring) with---or possibly even without---any type of financial support. Some programs are recommending that applications be submitted only for Fall. Applications submitted to these programs for Winter and Spring can not be processed for the following Fall term, applicants must reapply. Application processing fees submitted for these mid-year terms will not be refunded. Before you apply for either Winter or Spring, it is recommended that you check the program website or contact that program directly for additional information.
